Observation

  • There are 2 passengers whose Embarkation Port (Embarked) is NaN
  • There are 177 passengers whose age is not given in the data set
  • 204 people were staying in Cabin

handling missing values

1) Embarkation Port:
There are 2 passengers whose Embarkation Port (Embarked) is NaN

No analysis is done involving Embarkation Port and these 2 passengers have all others data. So 2 Nan values in Embarkation Port has no effect on our analysis.

2) Age:
There are 177 passengers whose Age is Nan.

a) When we analyse the correleation between 'Age' & 'Fare', we didn't consider the Passengers with Age as Nan

b) Pandas dataframe functions like mean(), groupby() ignore NaN Reference: http://pandas.pydata.org/pandas-docs/stable/missing_data.html

c) matplotlib.pyplot function plot(kind='hist') drops NaNs reference: http://pandas.pydata.org/pandas-docs/version/0.18.1/visualization.html

So missing values in Age are not filled using fillna()

3) Cabin:
There are 687 NaN values in Cabin. NaN value in Cabin is treated as the passenger didn't have any Cabin.

We added a column "isCabinPassenger" with boolean values in titanic_data for Analysis in Exploration phase.

def correlation(x, y):
    '''
    Fill in this function to compute the correlation between the two
    input variables. Each input is either a NumPy array or a Pandas
    Series.
    
    correlation = average of (x in standard units) times (y in standard units)
    
    Remember to pass the argument "ddof=0" to the Pandas std() function!
    '''
    std_x = (x - x.mean()) / x.std(ddof=0)
    std_y = (y - y.mean()) / y.std(ddof=0)
    return (std_x * std_y).mean()

Conclusion

I wanted to find out the factors that made people more likely to survive when titanic was sinking. So I calculated the survival percentage for male, female, children & elder passengers.

Analysis-1: Who Survived Male or Female?

We know that women, children & elder people get priority in case of disaster, I wanted to find out if they were rescued first in case of titanic.

From the Bar Plot in Fig 1, we can see that the percentage of survival in female passenger is 74.20% where the percentage of survival among male passenger is only 18.89%.

Survival percentage of female passenger is approximately 4 times than the male passenger. So we can infer that female passenger were rescuded as titanic was sinking.

From Fig-1, we can also infer that children (below 16 years) were rescued but Elders might not be rescued.

Analysis-2: Did passengers with better socio-economic status (SES) get privilege?

As we see in our society that people with better SES get more privilege. I wanted to find if that also held true during the rescue operation of Titanic. I calculated the survival percentage of each class of passenger to investigate.

From Fig 2, we can see that percentage of survival of Upper Class is 62.96%, Middle Class is 47.28% & Lower Class is 24.23%

Thus we can infer that people in Upper Class were survived more than the people of Middle and Lower Class

Analysis-3: Did passengers with relatives survived more than the passengers without relative?

From Fig -4, we can see that the percentage of Survival is more for passengers with one or more relatives compared to those who were alone.

Analysis-4: Is there any correlation between the Age and socio-economic status (SES)?

As we see in our society that socio-economic status improves with age. I wanted to find if that was true with Titanic passengers. So I calcutaled the correlation coefficient between Age & Fare.

The correlation coefficient between Age & Fare is 0.73 that indicates that they are positively correlated.

I plotted a scatter plot in Fig 5 to investigate the correlation between Age & Fare.

Not able to analyze: Age Vs Survived

I wanted to find if there is any correlation between age and survival? But I couldn't establish that from the dataset.
There is no significat difference between the mean age of Survived & non-Survived passengers. From the analysis it seems that gender (female), class, relatives in the ship helped people to survive.
